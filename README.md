# 🍅 トマト育てて食べまくり！

ブラウザだけで遊べる、トマト栽培シミュレーションゲームです。
立派なトマトを育てて収穫し、**たくさん食べる**ことがゴール！

## 遊び方

`index.html` をブラウザで開くだけ（インストール・ビルド不要）。

```
# 例: ローカルで開く
open index.html        # macOS
xdg-open index.html    # Linux
# もしくは簡易サーバ
python3 -m http.server
```

## ゲームの目的

トマトを育てて収穫し、**合計 20 個 食べたらクリア！**

## お世話のしかた

| アクション | 内容 |
|---|---|
| 🌱 タネをまく | 空き畑にタネをまいて栽培スタート |
| 💧 水やり | 水分を補給。少なすぎると弱り、多すぎると根腐れ・実割れ |
| ✂️ 脇芽かき | 脇芽を放置すると栄養を奪われ成長が遅くなる |
| 💩 追肥 | 肥料が切れると実が育たない |
| 🍃 葉かき | 収穫後に古い葉を整理。風通しを良くして病気・虫を予防 |
| 🐛 虫取り | 虫がついたら駆除して体力低下を防ぐ |
| 🥅 ネット | 完熟トマトを鳥から守る |
| 🍅 収穫 | 完熟した実を在庫へ |
| 🍴 食べる | 在庫のトマトを食べる（=ゴールの数を稼ぐ） |

## 育成ステージ

タネ → 発芽 → 苗 → 開花 → 着果 → 完熟 🍅

## ランダムイベント

- 🐛 **虫害**：葉が乱れていたり体力が低いと発生しやすい
- 🐦 **鳥害**：完熟トマトが食べられる（ネットで防げる）
- ⛈️ **大雨**：水分が急増して実が割れることがある

毎日コツコツお世話して、トマトを食べまくろう！
